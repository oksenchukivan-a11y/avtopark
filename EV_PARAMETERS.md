# Параметри для електричок (Kangoo Z.E. / Nissan Leaf) — що можна витягнути

Джерело: Teltonika OEM EV parameters + flespi teltonika protocol. Трекер FMB003 підтримує EV-параметри.

## 🔋 БАТАРЕЯ / EV (головне — замість палива)
| flespi-параметр | AVL ID | Що це |
|---|---|---|
| `can.vehicle.battery.level` | 411 | **Заряд батареї SoC, %** — це «бак» електрички |
| `can.vehicle.remaining.range` | 755 | **Запас ходу, км** |
| `can.vehicle.battery.charging.status` | 410 | **Статус зарядки** (заряджається / ні) |
| `can.vehicle.battery.power.consumption` | 412 | Споживання, кВт |
| `can.vehicle.battery.health` | 1151 | **Здоров'я батареї SoH, %** (знос) |
| `can.vehicle.battery.temperature` | 1152 | Температура батареї |

**Глибокі дані під час зарядки** (коли FMB ловить вікно зарядки):
напруга HV-батареї, струм, мін/макс темп. комірок, доступна енергія, лічильник повільних/швидких зарядок.

## 📍 ДЕ АВТО (універсальне, є завжди)
position.latitude/longitude, altitude (висота), speed (швидкість), direction (курс), satellites, hdop/pdop (точність), position.valid

## 🚗 СТАН / РУХ
- ignition status (запалювання/увімкнено)
- movement (рух так/ні)
- trip / odometer (GNSS-пробіг трекера)
- `can.vehicle.mileage` — одометр з машини (якщо EV віддає)
- towing (буксирування), crash (удар), excessive idling

## 📏 ПРОБІГ
- одометр машини `can.vehicle.mileage` (REB-стійкий) — головне
- GNSS-одометр трекера (запас)

## 📡 ТРЕКЕР / ЗВ'ЯЗОК
- battery voltage (внутрішня батарея трекера) + статус заряду
- external.powersource.voltage (бортова напруга)
- GSM signal level, оператор, IMEI, uptime
- температура трекера

## 🛡️ БЕЗПЕКА / ПОДІЇ
- GNSS jamming / GPS глушіння (РЕБ!) — детекція
- unplug detection (вийняли трекер)
- crash detection (удар/ДТП)
- geofence (заїзд/виїзд із зони)

---

## ⚠️ ВАЖЛИВО для налаштування EV-трекера
1. **Обов'язково режим «Non-OBD Compliant»** у Configurator (OBD II розділ) — інакше EV-батарея НЕ читається. Для бензину/дизеля було «OBD auto», для електрички — **Non-OBD Compliant**.
2. **VIN ввести вручну** (Manually) — рекомендація Teltonika для EV.
3. Увімкнути **EV Charge Monitoring** (детекція зарядки).
4. Сумісність: **Renault Kangoo Z.E. — Є у списку підтримуваних Teltonika FMB003** ✅. Nissan Leaf — підтвердити на тесті.
5. Нюанс спільноти: заряд (SoC) часто передається **лише під час руху** (як одометр Master — лише при заведеному).

## Як показувати в застосунку (режим електрички)
Замість «паливо в літрах»:
- 🔋 **Заряд батареї, %** (SoC) — головна цифра
- 🛣️ **Запас ходу, км**
- ⚡ **Статус зарядки** + журнал зарядок (стрибки заряду вгору = «заправка електрикою»)
- 🌡️ температура батареї, 🩺 здоров'я (SoH)
- + GPS, пробіг, швидкість (як у всіх)

«Заправка/злив палива» для EV → **«сесії зарядки»**: коли заряджали, з якого % до якого, скільки додали.
