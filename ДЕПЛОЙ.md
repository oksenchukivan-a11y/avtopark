# Автопарк — застосунок (деплой)

## 🌐 Адреса застосунку
**https://oksenchukivan-a11y.github.io/avtopark/**

## Репозиторій
github.com/oksenchukivan-a11y/avtopark (публічний, лише код)

## Як оновлювати код
1. Правлю файли локально в `07_Додаток_Автопарк/`
2. `git add -A && git commit -m "..."`
3. push з токеном: `git push https://ТОКЕН@github.com/oksenchukivan-a11y/avtopark.git main`
   (токен у `github_token.txt`; remote навмисно без токена)
4. Через ~1 хв оновлення живе на адресі вище

## Що показує (v1)
- карта всіх машин онлайн
- картка: паливо в літрах, пробіг за сьогодні (з одометра), швидкість, одометр
- деталі: пробіг Сьогодні/Вчора/Місяць (одометр), заправки 🟢 / зливи 🔴 (48 год)

## Конфіг
- ємності баків — у `app.js`, об'єкт `TANKS` (зараз тільки Audi 8436812 → 70 л)
- додавати машини автопарку сюди по мірі перенесення на flespi

## Токени (безпека)
- flespi-токен — вводиться в самому застосунку на телефоні, зберігається в браузері телефона (localStorage), у репозиторій НЕ потрапляє
- github_token.txt — можна видалити після деплою (або лишити для оновлень)
