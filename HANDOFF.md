# Автопарк — довідка для продовження (HANDOFF)

PWA для моніторингу автопарку поверх **flespi.io** (безкоштовний telematics backend).
Показує: паливо в літрах, пробіг з одометра, заправки/зливи, трек із зупинками, карту Google.
Розгорнуто на GitHub Pages. Мова інтерфейсу — українська.

## Застосунок
- **URL:** https://oksenchukivan-a11y.github.io/avtopark/
- **Файли:** `index.html` (UI+CSS), `app.js` (вся логіка), `sw.js` (PWA, network-first), `manifest.json`, `icons/`
- Без збірки — статичні файли. **Деплой = git push у `main`** → GitHub Pages оновлює за ~1 хв.
- Логін у застосунку: **flespi-токен** (вводиться один раз, зберігається в localStorage телефона; у репо НЕ зберігається).
- При зміні коду піднімати версію кешу в `sw.js` (`avtopark-vN`), щоб оновлення підхопилось.

## flespi backend
- **Канал** (приймає трекери Teltonika FMB003): host `ch1399442.flespi.gw`, port `38533`, TCP.
- **Пристрої** створюються через API: `POST /gw/devices`, `configuration.ident`=IMEI, `device_type_id`=19 (teltonika).
- **Конфіг авто — у МЕТАДАНИХ пристрою flespi (НЕ в коді):**
  ```
  metadata: { tank:<літри>, vehicle:"повна назва", icon:"<emoji>", color:"#hex", short:"коротка" }
  ```
  Застосунок читає метадані → бак/іконка/колір/підпис. **Нова машина = створити device + metadata, код не чіпати.**

## Як рахується (app.js)
- **Паливо в літрах:** `fuel.liters` (плагін) АБО `can.fuel.volume` (літри напряму — Master) АБО `can.fuel.level`% × tank (Audi).
- **Пробіг:** `can.vehicle.mileage` (одометр, стійкий до РЕБ) — головне джерело. GPS-трек окремо (під РЕБ замерзає → показуємо «⚠ РЕБ»).
- **Зупинки ≥3 хв:** швидкість ~0 **І** одометр не росте (інакше це рух під глушінням, не стоянка).
- **Заправки/зливи:** стрибки палива (в літрах); злив = різке падіння при зупинці.

## Кнопка «Перезавантажити трекер»
- `POST /gw/devices/{id}/commands-queue` body `[{"name":"custom","properties":{"text":"cpureset"}}]`
- Через чергу — виконається коли трекер на зв'язку (для зависань).

## Токени / приватні дані (НЕ в репо)
- **flespi-токен** — потрібен для API; користувач надає окремо в сесії. НЕ комітити.
- **git push** у цей репо — через інтеграцію хмарної сесії (github-токен не потрібен).
- **IMEI / телефони SIM / device id** — не зберігаємо в публічному репо. Брати з flespi:
  `GET /gw/devices/all`, `GET /gw/channels/1399442/connections/all`.

**🔑 ДИСТАНЦІЙНЕ додавання решти (БЕЗ зняття трекера):**
В OVERSEER → трекер → Команди → «00. Довільна команда» надіслати:
```
setparam 2007:ch1399442.flespi.gw;2008:38533;2009:0;2010:2
```
(2007=домен, 2008=порт, 2009=протокол TCP, 2010=режим 2/Duplicate; головний OVERSEER не чіпає). Потім «02. CPURESET».

**Залишилось додати:**
- Renault Master 9216 білий — бак ~80
- Renault Kangoo 2013 — бак ~60
- Renault Kangoo Z.E. — електро → заряд батареї (в списку Teltonika підтримується повноцінно)

Процес: надіслати команду в OVERSEER (або фізично Configurator) → зявиться на каналі → створити device+metadata → в застосунку.

## Незакрите
- **Калібрування палива Master:** перевірити, чи `can.fuel.volume` = реальні літри (тест: заправка +40 л, звірити з фото бака/чека; база була 22 л → мало стати ~62 л).
- **Kangoo Z.E.:** показ заряду батареї замість палива.
- Можливі майбутні: плеєр треку, графіки палива/швидкості, експорт, Telegram-сповіщення про злив.
