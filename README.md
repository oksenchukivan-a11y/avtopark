# Автопарк — власний моніторинг

Легкий PWA-застосунок для моніторингу автопарку через flespi.
Показує: паливо в літрах, пробіг з одометра (не GPS), заправки і зливи, карту машин онлайн.

- Статичний (без збірки), ставиться на телефон як додаток (PWA).
- Дані — з flespi REST API (токен вводиться один раз, зберігається в браузері).
- Хостинг — GitHub Pages (безкоштовно).
